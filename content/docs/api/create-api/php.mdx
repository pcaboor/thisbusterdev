```sh
composer create-project --prefer-dist laravel/laravel api-php
```

### Basic API code in PHP

Create an API `controller` with an articles resource:

```sh
php artisan make:controller ArticleController
```

In `ArticleController.php` : 

```php
<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class ArticleController extends Controller
{
    private $articles = [
        ['id' => 1, 'title' => 'PHP API', 'content' => 'Introduction Ã  Laravel']
    ];

    public function index()
    {
        return response()->json($this->articles);
    }

    public function store(Request $request)
    {
        $newArticle = $request->all();
        $this->articles[] = $newArticle;
        return response()->json($newArticle, 201);
    }
}
```

In `api.php` : 

```php
Route::get('/articles', [ArticleController::class, 'index']);
Route::post('/articles', [ArticleController::class, 'store']);
```
